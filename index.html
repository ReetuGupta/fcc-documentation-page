Introduction to JavaScript

JavaScript is a high-level, interpreted programming language that allows you to implement complex features on web pages — from dynamic content updates to animations and user interactions. It’s one of the core technologies of the web, alongside HTML and CSS.

Developed by Netscape in the 1990s, JavaScript has evolved into a versatile language used for frontend, backend, game development, and more. With the rise of frameworks like React, Angular, and Node.js, JavaScript now powers full-stack applications. 

                
                    console.log("Hello, JavaScript!");
  • JavaScript runs in the browser and on servers using Node.js.
  • It is dynamically typed and supports event-driven programming.

Variables and Data Types

Variables in JavaScript are containers for storing data values. You can declare them using var, let, or const. The choice affects scope and mutability. let and const are block-scoped, while var is function-scoped.

JavaScript supports various data types, including primitive types like string, number, boolean, undefined, null, and symbol, and non-primitive types like objects and arrays. 

                
                    let name = "Reetu";
                    const age = 20;
                    var isFrontendDev = true;
  • Use const for values that don't change.
  • Arrays and objects are reference types.

Functions

Functions in JavaScript are reusable blocks of code that perform specific tasks. They help in organizing code and improving modularity. You can declare functions using the function keyword, function expressions, or arrow syntax.

Functions can take parameters, return values, and even be stored in variables or passed as arguments, making JavaScript a functional programming language as well. 

                
                    function greet(name) {
                        return `Hello, ${name}!`;
                    }
                    console.log(greet("Reetu"));
  • Arrow functions (=>) provide a concise syntax.
  • Functions can be anonymous or named.

DOM Manipulation

The Document Object Model (DOM) is a programming interface for web documents. JavaScript can manipulate the DOM to update content, structure, and styles dynamically without reloading the page.

Using DOM methods like getElementById, querySelector, and addEventListener, you can make your websites interactive by responding to user actions like clicks and inputs. 

                
                    document.getElementById("btn").addEventListener("click", () => {
                        alert("Button clicked!");
                    });
  • querySelector is more flexible than getElementById.
  • Events help add interactivity to elements.

ES6 Features

ES6 (ECMAScript 2015) introduced major enhancements to JavaScript, making it more powerful and cleaner to write. Some key features include let/const, arrow functions, template literals, destructuring, and classes.

These features improve code readability and offer better control over scope, object handling, and iteration. ES6 also made JavaScript more modular with the import/export system. 

                
                    const user = { name: "Reetu", role: "Frontend Dev" };
                    const { name, role } = user;
                    console.log(`${name} is a ${role}`);
  • Use template literals with backticks for cleaner strings.
  • Destructuring makes it easy to extract values from objects and arrays.
